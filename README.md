# lin-sheet
基于 canvas 的表格编辑器。

![image](https://raw.githubusercontent.com/Liuzln/lin-sheet/master/gif/GIF%202020-4-8%2012-52-36.gif)

## 优化项清单
- [x] canvas 离屏渲染
- [x] 表格滚动时，从canvas离屏渲染的数据中裁剪所需区域，绘制到可视canvas上
- [ ] canvas 离屏渲染可视部分
- [ ] 更易用的鼠标点击控制光标
- [ ] 火狐浏览器在高DPI的情况下，滚动条无法跟着鼠标
